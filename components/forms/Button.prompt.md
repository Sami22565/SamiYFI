Primary call-to-action button — use `primary` for the main apply/submit action, `outline`/`ghost` for secondary actions.

```jsx
<Button variant="primary" size="lg" onClick={handleApply}>Apply Now</Button>
```

Variants: `primary` (ember, glow shadow), `secondary` (gold), `outline` (bordered), `ghost` (text-only). Sizes: `sm`, `md`, `lg`. Pass `disabled` to grey it out.
