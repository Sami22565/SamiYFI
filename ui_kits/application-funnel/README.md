# Application Funnel — UI Kit

A single-page apply funnel for Sami Yafeai's 1-on-1 fitness coaching mentorship, structurally modeled on a
common creator-mentorship funnel pattern (video hero → social proof → process → FAQ → multi-step apply form).
Built from scratch — no branding, copy, or imagery was copied from any reference site.

Sections: `HeroSection` (video/photo slot + headline + CTA), `StatsSection`, `HowItWorksSection`,
`TestimonialsSection`, `FAQSection`, `Footer`, and `ApplyModal` (3-step application form with confirmation state).

Open `index.html`, click "Apply Now" to walk through the form.
